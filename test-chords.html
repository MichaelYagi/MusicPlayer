Manual Test Instructions

  1. Open index.html in browser
  2. Click on "Grid Editor" tab
  3. Look at the Melody Pattern Grid
  4. You should see a chord item: "[chord: 3 notes]"
  5. Click on the chord header to expand it
  6. You should see 3 note inputs with note, duration, volume, and instrument fields
  7. Try editing the note values
  8. Try clicking "+ Add Note" button
  9. Try clicking "×" to remove a note
  10. Switch back to JSON Editor tab to see if changes are reflected
