Manual Test Instructions

  1. Open index.html in browser
  2. Open browser developer console (F12)
  3. Click "Save Project" button
  4. Now you should see detailed validation logs:
    • "Validating beat item 0: {...}"
    • "Beat item 0 validation checks: {...}"
    • "Beat validation PASSED at index 0"
    • Similar logs for each item
    • Same detailed logs for melody items
  5. If any item fails, you'll see exactly why
  6. Look for the specific validation check that fails
